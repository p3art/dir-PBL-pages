課程架構課程進行方式：
1.系統思考法：系統講解天文定義及數據處理，分成觀察、理解、設計
2.星座圖設計實作：觀賞Youtube短片，體驗程式撰寫，學生設計實作，以電腦繪製星座圖
1. 成果發表：上台展示作品並分享創作心得
與其他課程內涵連繫縱向1.以STEAM為導向，融合學習天文數理
2.結合高中數學、資訊科技、地球科學，深化專案學習能力
橫向1.國文歷史：史記天官書、二十四史之天文志
2.學務處：校慶週展示學生作品，參展創客教育博覽會，小論文
教學方法或策略1. 觀賞Youtube相關短片 (英語發音[旁白], 英文字幕)
2. STEAM導向的專案學習 (天文學、數學座標變換、資訊科技、程式設計)
3. 指導學生繪製星座圖，培養實作與解決問題的能力
4. 上台發表，增進學生楷模學習和語言表達能力
